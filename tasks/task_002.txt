# Title: Generate Initial OpenAPI Specification
# Status: done
# Dependencies: 1
# Priority: high
# Description: Generate the initial OpenAPI specification from the MySQL database schema using the sql-openapi tool.
# Details:
1. Connect to the local MySQL database
2. Run the command:
```bash
sql-openapi generate \
  --database mysql://root@localhost/duv_local \
  --output duv-openapi.yaml
```
3. Verify the generated YAML file contains all database tables and relationships

# Test Strategy:
Validate the generated YAML file against the OpenAPI 3.0 specification. Ensure all database tables are properly represented in the generated specification.
