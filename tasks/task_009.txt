# Details:
Create a file named 'eventdetail.php' in the /api/v1/ directory:
```php
<?php
require 'helpers.php';
$db = connectToDatabase();
validateApiKey($db, false); // public access

// Required parameter
$eventId = $_GET['id'] ?? null;
if (!$eventId) {
  http_response_code(400);
  outputJson(['error' => 'Event ID is required']);
}

$query = "SELECT
  ID AS id,
  EventName AS name,
  CountryCode AS country,
  DATE_FORMAT(Startdate,'%Y-%m-%d') AS date,
  Distance AS distance_km,
  City AS city,
  Website AS website_url
FROM tevent
WHERE ID=?";
$stmt = $db->prepare($query);
$stmt->bind_param('i', $eventId);
$stmt->execute();
$result = $stmt->get_result();

if ($result->num_rows === 0) {
  http_response_code(404);
  outputJson(['error' => 'Event not found']);
}

outputJson($result->fetch_assoc());
```
Ensure field names match those defined in the OpenAPI specification.
