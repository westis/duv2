DUV Ultramarathon Statistics API (2.0)

Download OpenAPI specification:Download

License: Apache 2.0

Introduction

Suggestion for new version of the API for DUV Ultramarathon Statistics. This API provides access to data on ultramarathon races, editions, results, runner profiles and more. The data is sourced from the DUV (Deutsche Ultramarathon Vereinigung) website, which is a comprehensive resource for ultramarathon statistics and information.

Base URL

The Base URL for the API is https://statistik.d-u-v.org/api. All endpoints referenced in the documentation are relative to this base URL.

Authentication

This API is open and does not require authentication to access its resources. You can make requests to any endpoint without including an API key or other authentication credentials.

Races

Operations related to races

Get races

Retrieves a list of races (by EventID in current data model). You can filter the results based on various criteria such as date, country, and race type.

query Parameters
search
string

Optional search terms for finding races by race name (EventId), city, or other attributes.

dateFrom
string <date>

The start date for filtering races, in the format YYYY-MM-DD.

dateTo
string <date>

The end date for filtering races, in the format YYYY-MM-DD.

countries
Array of strings

Two-letter continent codes or three-letter IOC country codes to filter races. Multiple values allowed.

raceType
string
Enum: "fixedTime" "fixedDistance" "backyardUltra" "stageRace" "other"

Type of race.

raceSurface
string
Enum: "trail" "road" "track" "indoor"

Surface of the race.

distanceMinKm
number <float>

Minimum race distance for open-ended distance ranges (in kilometers).

distanceMaxKm
number <float>

Maximum race distance for open-ended distance ranges (in kilometers).

durationMin
integer

Minimum race duration for non-standard times (in hours).

durationMax
integer

Maximum race duration for non-standard times (in hours).

disciplines
Array of strings

Predefined categories for standard distances or durations, such as 50km, 100mi, 6h, 10d, etc. Multiple values allowed.

rankingEligible
boolean

If set to true, only races that are eligible for ranking are returned.

resultStatus
string

Status of the race results, such as complete (C), partial (P), postponed (O), cancelled (R), etc.

limit
integer <= 100
Default: 25

The maximum number of races to retrieve.

offset
integer
Default: 0

The index of the first race to retrieve in the result set for pagination.

orderBy
string
Enum: "date" "numberOfFinishers"

Order races by date or the number of finishers. If not specified, the default ordering is by date.

lang
string
Default: "en"
Enum: "en" "de" "fr" "es" "it" "zh" "ja"
Example: lang=en

Specifies the language of the response data. Supports standard ISO 639-1 language codes.
