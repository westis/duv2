Races

Operations related to races

Get races

Retrieve races based on specified criteria.

query Parameters
year
required
string

Year of the event. Use 'futur' for future events, 'past1' for the past year, or specify a particular year.

country
required
string

Country code for the events, use 'all' for events in all countries.

dist
required
string

Distance category for the events, use 'all' for all distances.

cup
string

Cup category for the events, optional.

rproof
required
string

Ranking eligibility. Use '0' for all, '1' for eligible, '2' for not eligible.

mode
string

Mode of display, use 'list' for list view and 'map' for map view.

norslt
string

Include events without results. Use '1' to include.

page
integer

Page number for pagination.

Responses

Production server

https://statistik.d-u-v.org/json/mcalendar.php
